yaml 配置文件

注释 # 开头
key: value # key名字value值  标准格式

key:字符串
value: 类型
    数值类型 1, 1.1
    字符串类型 kk "kk"
    布尔类型 true false
    列表类型 [1, 2, 3]
    key:
    - 1
    - 2
    - 3
    字典类型
    {key:value}
    key:
        subkey: value
        subkey: xxx

使用缩进表示层级关系，    2个空格（建议值）/4个空格  不允许使用tab键；缩进的空格数目不重要，只要相同层级的元素左侧对齐即可。
